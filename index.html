# The Office — Discord Activity Una oficina virtual 2D multijugador dentro de Discord. Inspirado en Gather.town. ## Concepto Tu equipo remoto tiene un espacio de presencia donde ver a tus colegas trabajar. Cada persona es un avatar personalizable que se mueve por zonas con música ambient. Cuando estás cerca de alguien, podés chatear. ## Live 🔗 **https://theoffice-production.up.railway.app** ## Tech Stack - **Backend:** Node.js + Express + WebSocket (ws) - **Frontend:** Canvas 2D (nativo, sin frameworks) - **Hosting:** Railway ## Quick Start ```bash # Clone git clone https://github.com/maxsarlijagit/theoffice cd theoffice # Install npm install # Run server npm start # Open browser open http://localhost:3000 ``` ## Deploy Push a GitHub → Railway hace deploy automático. ``` git push origin main ``` ## Architecture ``` theoffice/ ├── server/ │ └── index.js # Express + WebSocket server ├── client/ │ └── index.html # Canvas 2D client (standalone) ├── package.json └── README.md ``` ### WebSocket API | Evento | Direction | Datos | |--------|-----------|-------| | `join` | → server | `{ name }` | | `move` | → server | `{ x, y }` | | `chat` | → server | `{ text }` | | `welcome` | ← server | `{ playerId, world, zones, color }` | | `player-joined` | ← server | player object | | `player-moved` | ← server | `{ playerId, x, y, zone }` | | `player-left` | ← server | `{ playerId }` | | `chat` | ← server | `{ from, fromId, text, zone, timestamp }` | ### Zonas | Zona | Descripción | Música | |------|-------------|--------| | Open Office | Zona central, trabajo | Lo-Fi | | Focus Room | Silencio, concentración | Ninguna | | Cafetería | Social, breaks | Upbeat | | Arcade | Minijuegos | Enérgica | | Estudio | Presentaciones | Neutral | ### Proximidad - Radio de proximidad: **3 tiles** - Cuando estás a ≤3 tiles de otro jugador: - Aparece un indicador visual - Se habilita el chat - Se pueden ver los mensajes en tiempo real ## Roadmap ### M1 ✅ Completo -
